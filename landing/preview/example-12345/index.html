Teaser Preview

Dies ist ein Platzhalter für eine Restaurant-Teaser-Vorschau.

In der echten Anwendung würde hier der HTML-Code für eine spezielle Speisekarte angezeigt werden, die von Task B generiert wurde.

Die Teaser-Vorschau zeigt wie die endgültige Speisekarte aussehen würde, inklusive:

  • Neuem Layout und Formatierung
  • Modernisierten Preisangaben
  • Besseren Lesbarkeit
  • Verkaufsfördernden Elementen

Die Preview wird aus out/<slug>/preview.html generiert.
